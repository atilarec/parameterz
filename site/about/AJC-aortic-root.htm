Goal

To work through an aortic root z-score calculation according to Gautier et al., using example data.

Calculate the Z-Score

The z-scores are calculated according to the formula supplied in the text:

            
    Z = [ln(measured diameter) - ln(predicted mean diameter)]/(SD),
    Z = [ln(measured diameter) - (a.ln(x) + b)]/(SD)

Interestingly, when they get around to presenting the actual data (Table 2), they use a different version of the formula- the data for the aortic annulus for a male subject is reported as:

            
    Z = [ln(d) - 2.78–0.47 * ln(BSA)]/0.1

This example calculation assumes a male patient with a BSA of 1.00 m2 and a measured aortic annulus of 15.5 mm:

            
    Z = [ln(15.5) - 2.78-0.47 * ln(1.00)]/0.1    
    
    which resolves to
    Z = [2.74084 - 2.78-0.47 * 0]/0.1
    
    ...
    Z = [2.74084 - 2.78-0]/0.1  
    
    ...
    Z = [-1.03916]/0.1
    
    which resolves to
    Z = -0.3916

Calculate the Upper Limit of Normal

The formula for prediction of the upper limit of normal (ULN) has the form:


    e(b + 2SD)*xa

Working through this calculation with the same assumptions (bsa = 1.0 m2, male):

        
    y = 19.57 * (BSA)0.47
    
    which resolves to
    y = 19.57 * 1.00.47
    
    ...
    y = 19.57 * 1
    
    which resolves to
    y = 19.57
